# Global Cheat Sheet

## Logging
- `print(str(...))`
- `console.log(x)` (single arg)
- `f.printString(...)`

## Type helpers
- `bool/int/float/str`
- `vec3/guid/prefabId/configId/faction/entity`

## Collections
- `list('int', items)`
- `dict(...)`

## Scene and entities
- `player(1)` / `stage` / `level` / `self`
- `GameObject.Find(...)`

## Other
- `Math` / `Mathf` / `Vector3` / `Random`
- `send('signal')` + `onSignal`
- `setTimeout` / `setInterval`
